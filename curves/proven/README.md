## Methodology

1. **Invariant Computation**: Calculate BSD invariants using SageMath/Cremona database
2. **Formula Verification**: Check both sides of the BSD equation
3. **Lean Formalization**: Create Lean4 proof structure
4. **Certificate Generation**: SHA-256 hash + Ψ symbiotic signature

## Verification

To verify a curve locally:

```python
# Using the notebook
jupyter notebook curves/proven/notebooks/validate_389a1.ipynb

# Using Python directly
python3 -c "
import json
with open('curves/proven/proof.json') as f:
    proof = json.load(f)
print(f'Curve: {proof[\"curve\"][\"label\"]}')
print(f'Status: {proof[\"signature\"][\"status\"]}')
print(f'Hash: {proof[\"signature\"][\"hash_sha256\"]}')
"
```

## Adding New Curves

To add a new validated curve:

1. Create `lean/CurveXXX.lean` with formal definitions
2. Create `notebooks/validate_XXX.ipynb` with validation
3. Update `proof.json` with the new curve certificate
4. Update this README
